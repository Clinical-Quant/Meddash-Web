## Detailed Feature Map (Final)
### 1. Dashboard (`/dashboard`)
- **AI-Powered Search**: Natural language query for medical intelligence.
- **Traditional Search**:
    - `Therapeutic Area`: Dropdown selection.
    - `Keyword / Phrase`: Text input for narrow focus.
    - `Pharma Stock Tickers`: Yahoo Finance integration (up to 5 tickers).
- **My Watchlists**:
    - Create Modal: Name, Keywords, Track toggles (Trials, Publications, Guidelines).
- **Recommended For You**:
    - Feed of papers/guidelines with "Why this is relevant" justification.
    - Actions: Like, Dislike, Open.
- **KOL Hub**:
    - Profile Tracking: Count/Limit (e.g., 0/200).
    - Add KOL Modal: Name, Google Scholar Link, ResearchGate Link, Institutional Profile Link, Category (Group), Notes.
- **Research Trends Weekly**:
    - Sections: "Emerging Therapeutic Areas" and "Predicted Future Research Focus".
    - Data points: Description, Confidence Level (High/Med), Source Link (e.g., FDA, Nature Medicine).

### 2. Evidence Library (`/evidencelibrary`)
- Central repository for saved evidence.
- Stats: Total Saved, Trials, Publications, Guidelines.
- Search/Filter by keywords and tags.

### 3. AI Utilities
- **Citation Generator** (`/citationgenerator`): PubMed URL input to output AMA, Vancouver, or APA formatting.
- **Clinical Summary** (`/clinicalsummary`): DOI/PubMed input to generate an AI-driven summary.
- **Learn Keyword** (`/learnkeyword`): Deep dive into specific medical terms using 12+ databases (NCBI, EMA, FDA, etc.).
- **Clinical Trial Matching** (`/clinicaltrialmatching`): Advanced matching based on Diagnosis, Stage, Biomarkers, and custom criteria.

### 4. System & Settings
- **Credits**: "Evidence Actions" counter (shared across AI tools).
- **Settings** (`/settings`): Notification preferences (Daily Email Alerts), Email profile view.
- **Tour**: On-boarding walkthrough (Skipable).
